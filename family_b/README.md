# Family B future cross-family case

This directory contains the minimum retained build overlays and reproduction
configuration for a future classical-controller ↔ registered learned-controller
case. The current Motivation Study and default Family A bootstrap do not use
these files.

No historical narrative, campaign result, or experimental conclusion is
retained here. The board overlays, airframe, state-input shim, and installer
scripts were created for an earlier dependency state. Every asset must be
revalidated against the active dependency lock before it is built or used as
evidence.

The scripts are optional and must never be called by
`scripts/setup/bootstrap_family_a.sh`.
