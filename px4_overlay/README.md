# PX4 Overlay Map

Reproducible PX4 changes are stored in:

- `boards/px4/sitl/` for board overlays;
- `patches/px4/` for source patches;
- `scripts/install_*.sh` for generated/copied airframe, board, DDS, and shim installation;
- `scripts/build_px4_*.sh` for build entry points.

The ignored `external/PX4-Autopilot` worktree is a local dependency, not the canonical artifact.
