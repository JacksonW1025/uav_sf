# Tier 0.5 §2.3 Search Loop Smoke

Date: 2026-06-26

Scope: final Tier 0.5 smoke before full campaign. This is not a full campaign and not a Tier 1 scale baseline. All real runs used the same `scripts/m2_map_elites.py` evaluator/oracle path with SIH headless/lockstep, PX4 `3042f906`, and `mc_nn_control` mode 23 identity required.

## Code Path

- `scripts/m2_map_elites.py`
  - Added `--target-properties` presets: `auto`, `behavior`, `behavior-step`, `route-a-catastrophic`, `validation`, or a comma list.
  - Added `--subspace full|route-a-switching|steady-wind-physics`.
  - Added `--strategy map-elites|random`; random keeps the same eval entry, oracle, archive, and progress outputs, only skipping elite parent selection.
  - Added `progress.jsonl`: best-so-far quality, archive bins, QD-score, selected parent, source, and per-eval outcome.
  - Added ROS environment overlay for `/opt/ros/jazzy` plus `ros2_ws/install` so real evaluator finds `ros2` and `px4_msgs` in non-interactive container commands.
- `scripts/mcnn_gate3_position_error_probe.py`
  - If the task node exits nonzero, terminate PX4 immediately and report the task rc. This prevents a trigger timeout from being mislabeled as a PX4 shutdown failure.
- `scripts/smoke_2p3_search_loop.sh`
  - Runs Part 1/2/3 through one shared harness. `SMOKE_SKIP_BUILD=1` reuses the existing build; `SMOKE_MOCK=1` runs the synthetic evaluator for mechanical checks only.

## P5 Vacuity Check

`property_oracle.robustness_p5` marks `detected_step_without_settling_candidates` as vacuous only when a step is detected but no candidate samples exist in the settling horizon. If candidates exist, it computes hold-window margin and returns the minimum rho; a controller that never enters the band still yields negative rho and is not silently swallowed as vacuous.

Part 2 deliberately avoided step stimuli, so P5 is not part of the formal behavior target for this smoke.

## Conclusion

- Part 1: machine validation passed. The MAP-Elites loop reads differential fitness, updates archive/QD metrics, and selects high-quality elites on known P1/P2 route-A signal.
- Part 2: behavior-class first probe was negative for confirmed P4/P6/P7 findings at this budget, but showed real gradients and guided improvement.
- Part 3: random baseline uses the same harness and oracle through one strategy flag. This de-risks the Tier 1 baseline retrofit before campaign scale.
