# M2 Guided Search

M2 wraps the M1 differential runner in a MAP-Elites search loop. The M1 runner remains the evaluator: one theta is run once with classical Offboard and once with RAPTOR external mode 23, metrics are extracted from ULOG, then the four-quadrant oracle is computed.

This pass implemented the M2 machinery and ran a first small guided search on 2026-06-24. It found one raw `primary_bug` candidate, but that candidate did not pass the required confirmation reruns. Therefore the confirmed first batch is currently empty. This is recorded as the actual result; thresholds were not relaxed and no result was fabricated.

## M2.0 Preconditions

### Fixed Safety Envelope

All M2 runs use `config/m2_safety_envelope.json` instead of per-theta open-ended defaults:

| metric | threshold | reason |
|---|---:|---|
| tracking max | 4.0 m | above M1 safe anchors, below the 10 m infrastructure-failsafe case |
| tracking RMS | 2.0 m | above M1 safe anchors but catches sustained loss of tracking |
| final error | 1.5 m | mission-completion bound for small M2 setpoint amplitudes |
| roll/pitch max | 75 deg | divergence/flip guard inherited from M1 |
| angular rate max | 8 rad/s | divergence guard inherited from M1 |
| motor saturation ratio | 0.99 | only continuous saturation is unsafe; lower saturation is fitness |
| min AGL | 0.25 m | ground-contact guard |

Noise floor is also fixed: tracking RMS `0.04 m`, roll/pitch max `1 deg`, angular-rate max `0.02 rad/s`, plus matching small floors for other deltas. `m1_compare.py` subtracts these floors before scoring divergence.

### Classical Baseline Decontamination

`m1_metrics.py` now reads `failsafe_flags` and emits:

- `infrastructure_limited`
- `infrastructure_reasons`
- `offboard_control_signal_lost_in_window`
- `nav_states_in_window`
- `target_nav_state_fraction`

Normal `mode_req_offboard_signal` is not treated as failure evidence; it is true during healthy Offboard operation. Classical runs are excluded from `primary_bug` if they lose Offboard signal or spend less than 98% of the mission window in the target nav state.

Offline regression:

- `anchor_sine_5hz` classical remains safe and usable.
- `anchor_step_10m` classical is unsafe and infrastructure-limited via `offboard_control_signal_lost` plus nav-state exit.

## Theta Space

The search space is weighted by the M1 clipping result:

- A tier: physical mismatch and wind. This is the main budget.
- B tier: timing and switch state.
- C tier: small-amplitude high-frequency setpoints only.

MAP-Elites bins are `dominant disturbance family x severity bucket`, where families are `A_physical`, `A_wind`, `B_timing`, and `C_setpoint`, and severity buckets are `low`, `mid`, and `high`.

The first nine bootstrap candidates are fixed seed-bank candidates weighted toward A-tier physical mismatch and wind. This is MAP-Elites initialization, not an M3 random/grid baseline.
